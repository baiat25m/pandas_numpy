# Pandas & NumPy for MLOps Engineers

## Learning Path

Work through the files in this order:

### NumPy
| File | Topic |
|------|-------|
| `numpy/01_arrays_basics.py` | Arrays, dtypes, shapes |
| `numpy/02_indexing_slicing.py` | Indexing, slicing, masking |
| `numpy/03_math_operations.py` | Math, broadcasting, ufuncs |
| `numpy/04_reshaping.py` | Reshape, stack, split |
| `numpy/05_statistics.py` | Stats, random, distributions |

### Pandas
| File | Topic |
|------|-------|
| `pandas/01_series_dataframe.py` | Series, DataFrame basics |
| `pandas/02_reading_writing.py` | CSV, JSON, Parquet I/O |
| `pandas/03_indexing_filtering.py` | loc, iloc, boolean filters |
| `pandas/04_cleaning.py` | Missing data, duplicates, types |
| `pandas/05_groupby_aggregation.py` | GroupBy, agg, pivot |
| `pandas/06_merging_joining.py` | Merge, join, concat |
| `pandas/07_time_series.py` | Datetime, resample, rolling |

### MLOps Practice
| File | Topic |
|------|-------|
| `mlops/01_feature_engineering.py` | Encoding, scaling, feature creation |
| `mlops/02_data_validation.py` | Schema checks, drift detection |
| `mlops/03_pipeline_etl.py` | End-to-end ETL pipeline |
| `mlops/04_model_monitoring.py` | Metrics tracking, data profiling |

## How to use these files
Each file has:
- **THEORY** section: explains the concept
- **EXAMPLES** section: runnable code with comments
- **EXERCISES** section: problems for you to solve (answers at bottom)

Run any file: `python numpy/01_arrays_basics.py`
